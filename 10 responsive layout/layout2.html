미디어 쿼리 Media Query

미디어 쿼리는 화면 크기에 따른 각각의 속성 값을 지정하여 여러가지 화면을 구성하는 기술.

@media only all and (조건) {실행문}

  • @media: 미디어 쿼리가 시작된다!
  • only: 미디어 쿼리 구문을 해석하라는 명령어 (생략가능)
  • all: 미디어 쿼리를 해석해야 할 대상을 나타냄 (생략가능)
    • all: 모든 미디어 유형에서 사용
    • print: 인쇄 장치에서 사용
    • screen: 컴퓨터 스크린에서 사용
    • aural: 화면을 읽어 소리로 출력해주는 장치에 사용
    • tv: TV에서 사용
    • handheld: 손에 들고 다니는 장치에 사용
    • projection: 프로젝트를 위한 장치
  • and: 조건들을 연결해주는.. (생략가능)
